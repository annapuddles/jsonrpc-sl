Send a JSON-RPC notification as a link message.

Parameters
linkThe link number to send the message to.
methodThe name of the JSON-RPC method to call.
params_typeOne of JSON_OBJECT or JSON_ARRAY, indicating how the parameters list should be interpreted.
paramsA list of parameters for the JSON-RPC method. If params_type is JSON_OBJECT, this should be a strided list of keys and values forming a named parameter list. If params_type is JSON_ARRAY, then this is a flat list of positional parameters.
